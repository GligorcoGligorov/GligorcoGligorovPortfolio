## Installation & Setup

1. **Clone the repository**
```bash
git clone https://github.com/GligorcoGligorov/GligorcoGligorovPortfolio.git
cd GligorcoGligorovPortfolio
```

2. **Install dependencies**
```bash
npm install
```

3. **Run locally**
```bash
npm start
```

The site will open at `http://localhost:3000`

4. **Build for production**
```bash
npm run build
```

## Deploying to Vercel

### Method 1: Using Vercel CLI

1. Install Vercel CLI
```bash
npm install -g vercel
```

2. Deploy
```bash
vercel
```

3. Follow the prompts and your site will be live!

### Method 2: Using Vercel Dashboard

1. Go to [vercel.com](https://vercel.com)
2. Sign in with GitHub
3. Click "Add New Project"
4. Import your GitHub repository
5. Vercel will auto-detect React and deploy!

## Customization

To customize your portfolio:

1. **Update personal info in App.jsx:**
   - Contact information
   - Projects
   - Skills
   - Experience

2. **Modify colors in App.css:**
   - Change CSS variables at the top of the file
   - Adjust color scheme to your preference

3. **Add your CV:**
   - Place your CV PDF in the `public` folder
   - Update the download link in the contact section
